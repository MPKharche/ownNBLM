# ownNBLM User Guide - Immersive Reading Mode

## Welcome to Your Learning Platform

ownNBLM transforms how you interact with documents by combining AI-powered answers with your personal annotations. This guide shows you how to make the most of the immersive reading experience.

---

## Getting Started

### 1. Add Your Documents

**Step 1: Open Sources Panel**
- The left sidebar contains your document sources
- If hidden, hover near the left edge of screen to reveal
- Click the **+** button to add a new source

**Step 2: Add a Folder**
- Label: Give it a memorable name (e.g., "Research Papers")
- Path: Select the folder containing your documents
- Click **Add Source**

**Step 3: Wait for Indexing**
- The system automatically scans and indexes your files
- PDF, Markdown, DOCX, and TXT files are supported
- You'll see a progress indicator during indexing

### 2. Start a Conversation

**Default Mode: Corpus-Wide**
- Ask questions across ALL your documents
- Great for: "What do my documents say about X?"
- The AI searches your entire library

**Ask Your First Question**
- Type in the input box at the bottom
- Press Enter or click Send
- Watch the AI stream its response in real-time

### 3. Enter Immersive Reading Mode

**When to Use:**
- During active learning sessions
- When reading AI responses with citations
- For distraction-free focus

**How to Enter:**
- Click **"Enter Reading Mode"** in the footer
- Or automatically enters when you click a citation
- Sidebars collapse, floating controls appear

**What Changes:**
- Full-screen chat interface
- Minimal distractions
- Floating controls at top
- Sidebars hide (hover to reveal)

## Working with Documents

### Opening a Document

**From Citations:**
1. AI response includes numbered source citations
2. Click any citation chip to open that document
3. Screen splits: chat left, document right
4. Document auto-scrolls to the cited page

**Manual Opening:**
1. Expand Sources panel (left sidebar)
2. Click any indexed document
3. Document viewer opens on right side

### Navigating Documents

**PDF Navigation:**
- Previous/Next buttons for pages
- Page counter shows current position
- Zoom controls: 50% to 200%
- Click cited page number in chat to jump

**Markdown Documents:**
- Rendered with full formatting
- Syntax highlighting for code blocks
- Auto-scroll to cited line numbers
- Clean, readable typography

**Keyboard Shortcuts (Coming Soon):**
- `Arrow Up/Down`: Previous/next page
- `+/-`: Zoom in/out
- `Cmd/Ctrl + D`: Close document

## Adding Notes & Annotations

### Message Notes

**Purpose:** Capture insights from AI responses

**How to Add:**
1. Hover over any AI message
2. Click the **note icon** (appears in top-right)
3. Type your note in the input box
4. Press Enter or click Save

**Your notes:**
- Stay attached to that message
- Appear in yellow boxes below the message
- Export with the session
- Searchable (future feature)

**Use Cases:**
- "Remember to research this further"
- "Key concept for my paper"
- "Question to ask professor"
- "Compare with Smith et al."

### Session Notes Panel

**Access:**
- Right sidebar → Notes tab
- Shows all notes from current session
- Organized chronologically
- Click to jump to original message

### Document Annotations (Coming Soon)

**Features:**
- Highlight text in documents
- Add bookmarks to important pages
- Create standalone notes on documents
- All sync with chat citations

## Exporting Your Knowledge

### Export Entire Session

**What Gets Exported:**
- All messages (yours and AI's)
- Complete citations with page numbers
- Your notes and highlights
- Properly formatted markdown

**How to Export:**
1. Click **Export** button (top-right in reading mode)
2. Downloads as `.md` file
3. Open in any text editor or markdown viewer
4. Import to Notion, Obsidian, etc.

**Use Cases:**
- Study notes for exams
- Research documentation
- Knowledge base articles
- Share insights with team

### Copy Individual Messages

**Quick Copy:**
1. Hover over any message
2. Click the **copy icon**
3. Markdown-formatted text copied to clipboard
4. Paste anywhere

**What's Included:**
- Full message text
- All citations
- Your notes on that message

### External Tools

**The exported markdown works with:**
- Obsidian (personal knowledge base)
- Notion (collaborative docs)
- Roam Research (networked notes)
- Standard text editors
- GitHub/GitLab (documentation)
- Academic writing tools (LaTeX, Pandoc)

## Managing Sessions

### What Are Sessions?

**Two Types:**

**Corpus Sessions (Default)**
- Search across ALL documents
- Great for exploratory questions
- No document filtering

**Scoped Sessions**
- Limited to selected documents
- Focused, deep-dive conversations
- Prevents information overload

### Creating a Scoped Session

**Step 1: Open Sessions Panel**
- Right sidebar (hover if hidden)
- Click **Sessions** tab
- Click **New Session**

**Step 2: Configure**
- Give it a descriptive name
- Select specific documents to include
- Leave empty for corpus-wide search

**Step 3: Start Learning**
- Session appears in sessions list
- Switch anytime by clicking
- Each session has separate chat history

**When to Use Scoped Sessions:**
- Studying specific papers
- Comparing related documents
- Working on a focused topic
- Reducing noise from large corpus

## Sidebar Management

### Collapsible Sidebars

**Left Sidebar: Sources**
- Your document library
- File tree browser
- Index status

**Right Sidebar: Sessions & Notes**
- Switch between sessions
- View all your notes
- Manage annotations

### Toggling Sidebars

**When Hidden:**
- Hover near screen edge to reveal toggle button
- Click anywhere on trigger zone (1px wide)
- Or click the toggle button that appears

**When Visible:**
- Click chevron button to collapse
- Or hover and click toggle
- Smooth 300ms animation

**Pro Tip:** 
- Keep both collapsed for maximum reading space
- Reveal on-demand when needed
- Keyboard shortcuts coming soon

## Tips for Effective Learning

### 1. Ask Follow-Up Questions

**Don't stop at the first answer:**
- "Can you explain that in simpler terms?"
- "What are the implications of this?"
- "How does this relate to [concept]?"

**The AI has context:**
- Remembers the conversation
- Can dive deeper
- Makes connections

### 2. Use Citations Actively

**Every citation is a learning opportunity:**
- Click to read the original source
- Compare AI summary to source
- Look for additional context
- Add notes about differences

### 3. Build Your Note Collection

**Treat notes as your future self:**
- Write in your own words
- Note why it's important
- Add connections to other ideas
- Use consistent formatting

**Future You will thank you:**
- Notes are searchable (coming soon)
- Export creates study guides
- Build personal knowledge base

### 4. Create Focused Sessions

**For deep work:**
- One topic per scoped session
- Related documents only
- Clear naming convention
- Export when complete

### 5. Review Your Exports

**Periodic review strengthens learning:**
- Re-read exported sessions
- Consolidate related notes
- Identify knowledge gaps
- Plan next questions

## Troubleshooting

### Document Won't Open
- Check if indexing is complete
- Verify file format is supported
- Try re-scanning the source

### Citations Not Clickable
- May need to wait for response to complete
- Refresh page if streaming stalled
- Check document is still in library

### Notes Not Saving
- Ensure you clicked Save or pressed Enter
- Check browser localStorage not disabled
- In production, check backend connection

### Sidebar Won't Collapse
- Try clicking toggle button directly
- Refresh page to reset state
- Check browser console for errors
